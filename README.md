# Raptorex Controller Releases

Bu repo sadece Raptorex CNC Controller uygulamasinin kurulum dosyalarini icerir.

## Indirme

En son surumu indirmek icin [Releases](https://github.com/RefikCodes/Raptorex-Controller-Releases/releases/latest) sayfasini ziyaret edin.

## Kurulum

1. En son `RaptorexController-X.X.X-Setup.exe` dosyasini indirin
2. Kurulum dosyasini calistirin
3. Kurulum sihirbazini takip edin

## Lisans

Bu uygulama Raptorex CNC tarafindan gelistirilmistir.
