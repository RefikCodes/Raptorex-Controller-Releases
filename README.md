# Raptorex CNC Controller - Releases

## İndirme

Son sürümü [Releases](https://github.com/RefikCodes/Raptorex-Controller-Releases/releases/latest) sayfasından indirebilirsiniz.

### Dosyalar
- **RaptorexController_Setup_X.X.X.exe** - Windows Installer (önerilen)
- **RaptorexController_vX.X.X.zip** - Portable versiyon
